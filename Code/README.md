Saliency Detection

Detection of 'salient' objects in an image.


Requirements:

1. CMake 2.8+
2. OpenCV 3.0+


Procedure:

1. cd /path/to/BMS
2. cmake .
3. make
4. python demo.py

demo.py takes an image in the Input/ folder and writes the predicted eye-fixation map for that image in the Output/ folder.
